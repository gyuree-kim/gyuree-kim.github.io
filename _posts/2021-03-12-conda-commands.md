## 아나콘다 환경 생성
```
conda create -n {env name} python={python version}
```

## 생성된 환경 리스트
```
conda env list
```

## 특정 환경 삭제
```
conda env remove -n {env name}
```

## 환경 활성화
```
activate {env name}
```

# 환경 비활성화
```
deactivate
```
